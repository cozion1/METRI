# 🔧 3) חוקי זיהוי חכמים (Pattern Engine)

כך זה עובד בפועל:

### 🧩 חוק בדידות

אם מופיעים מילים:
שקוף / לבד / לא רוצים אותי / לא שמים לב
→ תחום = בדידות

---

### 🔥 חוק כעס

אם:
נמאס לי / אני מתפוצץ / בא לי להרביץ / שכולם ייעלמו
→ תחום = ויסות רגשי / כעס

---

### 😰 חוק חרדה

אם:
עדיף לשתוק / כולם שופטים / לא כדאי לנסות
→ תחום = חרדה חברתית

---

### 🧠 חוק ייאוש

אם:
אין טעם / זה תמיד ככה / בא לי להיעלם
→ תחום = ייאוש

---

## 🧠 1) מה הייתה המשימה שעבדנו עליה ב-BEZEN-HQ

BEZEN-HQ נבנה כ־**מטה המערכת**:

```
BEZEN_HQ/
├── agents/          ← הסוכנים (BEZEN-CORE)
├── office/
│   ├── content/     ← כל המאגרים שלך (משפטים, תכונות, תבניות)
│   ├── mind/        ← חוקים, סכמות, מבני חשיבה
├── runtime/         ← ריצות, לוגים, תוצאות
```

כלומר:

✔ כל הידע
✔ כל המשפטים
✔ כל המבנים הרגשיים
✔ כל חוקי החשיבה

כבר קיימים שם.

זה **המוח והספרייה** של BEZEN.

## 2) יוצרים “מנוע מינימלי” מקומי (בלי API)

בתוך:

```

```

תיצור קובץ בשם:


שים בו את הקוד הבא:

```python
# BEZEN_HQ/office/engine/engine_local.py
from pathlib import Path
import re, json
from datetime import datetime

ROOT = Path(__file__).resolve().parents[2]        # .../BEZEN_HQ
INPUT = ROOT / "office" / "inputs" / "teen_sentences.txt"
OUTDIR = ROOT / "runtime" / "logs"
OUTDIR.mkdir(parents=True, exist_ok=True)
OUT = OUTDIR / "engine_output.jsonl"

# חוקים בסיסיים (אפשר להרחיב בלי סוף)
DOMAIN_RULES = [
    ("loneliness", [r"\bלבד\b", r"שקוף", r"לא רוצים אותי", r"לא מבינים אותי", r"אין לי.*עם מי לדבר"]),
    ("self_worth", [r"לא שווה", r"אני אפס", r"כישלון", r"לא מספיק"]),
    ("social_anxiety", [r"יצחקו עליי", r"כולם שופטים", r"עדיף לשתוק", r"אני לא מעז"]),
    ("anger", [r"נמאס לי", r"מתפוצץ", r"בא לי לצעוק", r"להחזיר להם"]),
    ("despair", [r"אין טעם", r"תמיד יהיה ככה", r"בא לי להיעלם"]),
]

BALANCE_MAP = {
    "loneliness": ("נסיגה", "חיבור בטוח"),
    "self_worth": ("השוואה/מחיקת עצמי", "ערך פנימי"),
    "social_anxiety": ("הימנעות", "אומץ רגוע"),
    "anger": ("התפרצות", "ויסות רגשי"),
    "despair": ("ויתור/קיבעון", "תקווה פעילה"),
}

def severity(text: str) -> str:
    # חומרה פשוטה: מילים מוחלטות / קיצוניות
    if any(w in text for w in ["תמיד", "אף פעם", "אף אחד", "אין טעם"]):
        return "גבוהה"
    return "בינונית"

def detect_domain(text: str) -> str:
    for domain, pats in DOMAIN_RULES:
        for pat in pats:
            if re.search(pat, text):
                return domain
    return "unknown"

def bridge_sentence(domain: str) -> str:
    # משפט גשר אמין (לא “חיובי בכוח”)
    bridges = {
        "loneliness": "עכשיו אני מרגיש לבד, אבל זה לא אומר שככה זה תמיד.",
        "self_worth": "עכשיו אני מרגיש פחות, אבל זה לא מגדיר את מי שאני.",
        "social_anxiety": "עכשיו אני מפחד מתגובה, וזה טבעי כשחשוב לי מה יחשבו.",
        "anger": "עכשיו אני מוצף בכעס, ואני יכול לעצור רגע לפני תגובה.",
        "despair": "עכשיו קשה לי לראות מוצא, אבל התחושה הזו יכולה להשתנות.",
        "unknown": "עכשיו קשה לי, ובוא נפרק את זה צעד-צעד."
    }
    return bridges.get(domain, bridges["unknown"])

def helpful_sentence(domain: str) -> str:
    helpful = {
        "loneliness": "אני יכול ליצור חיבור קטן ובטוח, צעד אחד היום.",
        "self_worth": "הערך שלי לא תלוי בלייקים או בהשוואות.",
        "social_anxiety": "אני יכול לדבר בקטן ובבטוח, בלי להוכיח כלום.",
        "anger": "אני בוחר תגובה שמכבדת אותי, גם כשאני כועס.",
        "despair": "צעד קטן אחד עדיף על לוותר — והוא מספיק להיום.",
        "unknown": "אני יכול לבחור צעד קטן שמחזיר לי שליטה."
    }
    return helpful.get(domain, helpful["unknown"])

def main():
    if not INPUT.exists():
        raise FileNotFoundError(f"Missing input file: {INPUT}")

    lines = [ln.strip("• \t") for ln in INPUT.read_text(encoding="utf-8").splitlines()]
    sentences = [ln for ln in lines if ln and not ln.startswith("שלב") and not ln.startswith("משפטים")]

    with OUT.open("w", encoding="utf-8") as f:
        for s in sentences:
            domain = detect_domain(s)
            sev = severity(s)
            neg, pos = BALANCE_MAP.get(domain, ("לא ידוע", "איזון"))
            rec = {
                "ts": datetime.now().isoformat(timespec="seconds"),
                "input": s,
                "domain": domain,
                "severity": sev,
                "pattern": neg,
                "balanced_trait": pos,
                "bridge": bridge_sentence(domain),
                "helpful": helpful_sentence(domain),
            }
            f.write(json.dumps(rec, ensure_ascii=False) + "\n")

    print(f"✅ Done. Wrote: {OUT}")

if __name__ == "__main__":
    main()
```

# עכשיו מה עושים עם התוכן?

אני רואה אצלך שיש שורות עם:

* `domain: "unknown"`
* ויש `loneliness`, `anger`, `social_anxiety`, `self_worth`, `despair`

✅ השלב הבא במנוע:

1. לצמצם `unknown` כמעט לאפס
2. להוסיף לכל שורה גם:
   **balanced_trait + bridge + helpful** (כמו שכבר כתבת בקוד, אבל בקובץ רואים כרגע בעיקר domain/severity)

---

## משימה קצרה עכשיו (בלי בלבול)

תעשה אחד משני הדברים:

### אפשרות A (הכי קל):

תעתיק לכאן **שורה אחת** מתוך הקובץ שיש בה `domain: "unknown"`.

או

### אפשרות B:

תעתיק **2 שורות**: אחת `unknown` ואחת עם `self_worth`
ואני אגיד לך בדיוק מה לשנות ב־`engine_local.py` כדי שה־unknown יזוהה.

רוצה A או B?
מצוין. זה בדיוק מה שהייתי צריך.
השורה הראשונה כבר “יושבת” יפה. השורה השנייה נכנסה ל-`unknown` כי המשפט הוא **שייכות/קבלה חברתית** (חשש מדחייה), לא “כללי”.

להלן מה עושים כדי שהמנוע יהיה חד:

---

## 1) מה המשימה עכשיו

להעלים `domain: "unknown"` עבור המשפט:

> "אני לא יודע אם באמת רוצים אותי פה"

ולסווג אותו לדומיין נכון + לתת לו `pattern` נכון + `balanced_trait` נכון.

---

## 2) הפתרון (מדויק)

### א. תחליף את הדומיין ל:

**`belonging`** (שייכות)
או אם אתה רוצה להישאר עם מה שכבר יש לך: **`social_anxiety`**.

אני ממליץ: `belonging` כי זה לא חרדה “חברתית” קלאסית, אלא ספק בשייכות וקבלה.

### ב. תחליף `pattern`

מ־`לא ידוע` ל:
**`חשש מדחייה`** / **`בדיקת שייכות`**

### ג. תחליף `balanced_trait`

מ־`איזון` ל:
**`ביטחון שקט`** / **`שייכות פנימית`**
(אני מעדיף “ביטחון שקט” לנוער)

---

## 3) דוגמה לתוצאה הרצויה (השורה המתוקנת)

תוכל ממש להחליף את השורה השנייה לזה:

```json
{"ts":"2026-01-30T10:00:11","input":"אני לא יודע אם באמת רוצים אותי פה","domain":"belonging","severity":"בינונית","pattern":"חשש מדחייה","balanced_trait":"ביטחון שקט","bridge":"יש בי עכשיו ספק אם אני רצוי, ואני יכול לבדוק עובדה אחת במקום לנחש.","helpful":"אני שווה גם כשאני לא בטוח—ואני בוחר צעד קטן שמחזיר לי קרקע."}
```

---

## 4) הצעד הבא בפועל (כדי שהמנוע יעשה את זה אוטומטית)

עכשיו צריך להוסיף במנוע **חוק זיהוי** למשפטים עם מילים כמו:

* “רוצים אותי”, “לא רוצים אותי”, “לא שייך”, “לא מתאים”, “דוחים אותי”, “לא סופרים אותי”, “מיותר פה”

### בדיוק מה להוסיף בקוד:

ב־`engine_local.py` יש לך כנראה פונקציה שמחזירה `domain` לפי מילות מפתח (או `detect_domain()` / `classify_domain()`).

תוסיף כלל כזה:

* אם המשפט כולל: `"רוצים אותי"` או `"רוצים אותי פה"` או `"לא רוצים אותי"` או `"שייך"` או `"דוחים אותי"`
  → תחזיר `"belonging"`

ואז תוסיף ב-`BALANCE_MAP`:

* `"belonging": ("חשש מדחייה", "ביטחון שקט")`
  (או איך שהמפה שלך בנויה)

---

## 5) בדיקה חדה (כדי שלא תיפול שוב ל-unknown)

אחרי שאתה מוסיף את הכלל:

1. תריץ שוב:

```powershell
python office/engine/engine_local.py
```

2. תפתח את `runtime/logs/engine_output.jsonl`
3. תוודא שהשורה של המשפט הזה יצאה עם:

* `domain: belonging`
* `pattern: חשש מדחייה`
* `balanced_trait: ביטחון שקט`
