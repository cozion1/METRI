# BEZEN CODEX CORE (Unified)

This folder is prepared for OpenAI Codex CLI / agent-style coding assistants.

## How to use
- Keep this repo as your knowledge pack.
- Add new traits as folders under `traits/` with the same 1..6 files.
- Ask Codex to: (a) detect trait from user sentence, (b) build a scene using the matching folder.
