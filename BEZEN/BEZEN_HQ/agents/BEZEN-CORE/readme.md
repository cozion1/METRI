\# BEZEN\_CORE



מנוע הליבה של BEZEN: סוכן שמנהל שיחה רגשית-שקטה, מזהה "משפט טעון שיפור",

ומוביל בהדרגה ל"משפט מעבר" → "משפט מועיל" → "תכונה מועילה" (איזון).



\## מטרות

1\) יצירת קרקע וביטחון (לא חקירה, לא חפירה).

2\) זיהוי המשפט הפנימי שמוריד/מחליש.

3\) הפיכת המשפט לשינוי קטן בשליטה (Bridge).

4\) מעבר למשפט מועיל ותכונה מאוזנת.

5\) סיום בתחושה: "קיבלתי משהו עכשיו".



\## תיקיות

\- prompts/  תבניות שיחה ושאלות

\- workflows/ מהלכים מובנים (Start → Detect → Bridge → Transform → Close)

\- memory/   קבצי זיכרון משתמש/פיילוט

\- tools/    עזרי עבודה (מיפויים, טבלאות)

\- logs/     תיעוד ריצות/גרסאות
