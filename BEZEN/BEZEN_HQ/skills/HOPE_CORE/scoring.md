# HOPE_CORE — Scoring

## Goal
להגיע ל־100 נקודות כדי לעבור ל־Skill הבא.

## How to earn points
- כל מיני־משחק נכון: +25 נקודות
- מיני־משחק לא נכון: 0 (אפשר לנסות שוב בסיבוב הבא)

## Checkpoints
- 25% = אחרי מיני־משחק אחד נכון
- 50% = אחרי שניים
- 75% = אחרי שלושה
- 100% = אחרי ארבעה

## Promotion Rule
כשמגיעים ל־100 נקודות:
- המשתמש מקבל הודעת מעבר קצרה ומרגיעה
- מעבר אוטומטי ל־Skill הבא (שם יוגדר מאוחר יותר)
