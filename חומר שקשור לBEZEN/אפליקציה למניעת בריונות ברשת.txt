## 🌀 פרק 1 – "הסטורי"

**נושא:** אלימות רגשית ודיגיטלית (לעג, שיימינג, השפלה ברשת).
**ערך מרכזי:** חמלה עצמית ולקיחת אחריות.
**מטרה רגשית:** לזהות את הכאב – ולבחור תגובה מודעת במקום תגובה פוגעת.

## 🎮 מצב 1 – משחק סולו (משחק אישי מודרך)

**מסך פתיחה:**

> כותרת: *“הסטורי”*
> משפט פתיחה:
> “מישהו העלה עליך סטורי מעליב. כולם ראו. אתה רותח מבפנים.”
> שאלה: “מה עולה לך באותו רגע?”

**השחקן כותב** (לדוגמה): “אני מרגיש שכולם שונאים אותי.”

> **BEZEN-CORE:** “שמעתי משפט עם כאב אמיתי – ‘כולם שונאים אותי’. זה נשמע כמו רגש מציף.”
> “נפנה רגע ל־💖 *BEZEN-HEART*.”

**BEZEN-HEART:**

> “קח/י שלוש נשימות איטיות.
> שאיפה 4, החזקת אוויר 4, נשיפה 6.
> איזה רגש יש שם? בושה? כאב? כעס?”

**השחקן:** “כעס.”

> **BEZEN-HEART:** “הכעס שלך מגן על הלב שלך.
> זה סימן שאתה זקוק להרגיש שמכבדים אותך.
> רוצה לנסות משפט חדש שירכך את זה?”

**השחקן:** “כן.”

> **BEZEN-CORE:** “משפט מועיל שלך:
> 💬 *‘אני כועס, אבל אני לא חייב לפגוע כדי להרגיש חזק.’*
> איך זה מרגיש לומר את זה?”

**השחקן:** “קצת קל יותר.”

> **BEZEN-CORE:** “נפלא. זה רגע של שליטה מתוך חמלה.
> צעד קטן קדימה: כתוב לעצמך שם אחד של אדם בטוח שאפשר לדבר איתו עליו.”

**סיום:**

> “💎 אתה הרגע יצרת מרווח בין כאב לתגובה.
> זה המקום שבו אלימות נגמרת, ושלווה מתחילה.”

## 🧭 מצב 3 – מוד חונך / מנחה

**מטרת החונך:** להחזיק את המרחב בביטחון, בלי שיפוט.

**שלבים למנחה:**

1. **פתיחה רכה:**
   “היום ניגע בנושא אמיץ – אלימות ברשת. זה קורה, אבל כאן אנחנו נלמד איך לעצור את המעגל.”
2. **הפעלת המשחק הקבוצתי:**
   בחרו יחד תפקידים → אפשר להקרין את סצנת הסטורי → אפשר לשחק תפקידים.
3. **שאלות לעיבוד:**

   * “מה קורה בלב של כל צד?”
   * “מה לדעתכם עוזר לעצור את זה בזמן אמת?”
   * “איך זה מרגיש כשמישהו בוחר לא להגיב באלימות?”
4. **תרגול BE ZEN קצר:**
   (5 נשימות משותפות + משפט הרגעה.)
   “אני בטוח/ה. אני נושם/ת. אני לא חייב/ת לפגוע כדי להרגיש חזק/ה.”
5. **סגירה:**
   כל משתתף משתף משפט מועיל אחד שהוא לוקח איתו הביתה.

// packages/core/agentRouter.ts
export type Tag = "mind"|"heart"|"body"|"action"|"soul";
const riskWords = [/להרביץ|להרוג|לא רוצה לחיות|סכין|אאבד את/i];

export function detectRisk(text: string){ return riskWords.some(r=>r.test(text)); }

export function classify(text: string): Tag {
  const t = text.toLowerCase();
  if (/(אני לא שווה|אני לא ראוי|תמיד|אף פעם)/.test(t)) return "soul";
  if (/(שונאים אותי|בושה|פחד|כעס|עצבות)/.test(t)) return "heart";
  if (/(כואב לי|בטן|לחץ בחזה|דופק)/.test(t)) return "body";
  if (/(אני הורס|לא מצליח|שלחתי סטורי|צחקתי עליו)/.test(t)) return "action";
  return "mind";
}

export function craftAffirmation(tag: Tag, text: string){
  switch(tag){
    case "heart": return "אני כועס/ת, ואני בוחר/ת לא לפגוע.";
    case "mind":  return "אני רואה את הסיפור יותר מאוזן, צעד-צעד.";
    case "body":  return "אני נושם/ת 4-4-6 והגוף נרגע לאט.";
    case "action":return "אני בוחר/ת התנהגות שמכבדת אותי ואת האחרים.";
    case "soul":  return "אני ראוי/ה לכבוד ואהבה – גם כשכואב.";
  }
}

"use client";
import {useState} from "react";
import {classify, detectRisk, craftAffirmation} from "core/agentRouter";

export default function Chat(){
  const [log,setLog]=useState<{role:"user"|"bot",text:string}[]>([
    {role:"bot",text:"היי 🌿 מה קרה שם בסטורי? כתוב/כתבי במשפט אחד."}
  ]);

  async function onSend(msg:string){
    setLog(l=>[...l,{role:"user",text:msg}]);
    if(detectRisk(msg)){
      setLog(l=>[...l,{role:"bot",text:"שמעתי מילות סכנה. נעצור לרגע ונעבור למסך בטיחות עם משאבים בישראל."}]);
      // ניתוב ל/ safe
      window.location.href="/safe";
      return;
    }
    const tag = classify(msg);
    const affirm = craftAffirmation(tag,msg);
    const agentIntro = {
      mind:"נלך רגע ל-MIND (מסגור מחדש).",
      heart:"נלך רגע ל-HEART (נשימה ורגש).",
      body:"נלך רגע ל-BODY (קרקוע קצר).",
      action:"נלך רגע ל-ACTION (צעד-קטן).",
      soul:"נלך רגע ל-SOUL (אמונה מיטיבה)."
    }[tag];
    setLog(l=>[...l,
      {role:"bot", text:`זיהיתי שזה ‹${tag.toUpperCase()}›. ${agentIntro}`},
      {role:"bot", text:`💬 משפט מועיל: ${affirm}`}
    ]);
  }

  return (
    <div className="max-w-md mx-auto p-4">
      <div className="space-y-2 h-[60vh] overflow-y-auto border rounded p-3 bg-white">
        {log.map((m,i)=>(
          <div key={i} className={m.role==="bot"?"text-slate-700":"text-sky-700 text-right"}>
            {m.text}
          </div>
        ))}
      </div>
      <form onSubmit={e=>{e.preventDefault(); const f=e.target as any; const v=f.msg.value.trim(); if(v) onSend(v); f.reset();}}>
        <input name="msg" placeholder="כתוב/י כאן..." className="w-full mt-3 border rounded p-2"/>
      </form>
      <div className="mt-3 text-sm text-slate-500">נשימה מהירה: שאיפה 4 • החזקת אוויר 4 • נשיפה 6</div>
    </div>
  );
}

* **בטיחות**: סינון מילות סיכון בצד לקוח + שרת

## תרשים מאפיינים ל־MVP

* סצנה 1 “הסטורי”: סולו, קבוצתי (Room code), חונך.
* BEZEN-CORE: מיפוי משפט→סוכן (mind/heart/body/action/soul).
* חדר נשימה (טיימר 4–4–6).
* יומן משפטים מועילים.
* מסך בטיחות עם קישורי עזרה בישראל.
* ניתוב דגלי סיכון (מילים: התאבדות/אלימות/סכין/להרביץ…).

## שלד קוד (מוכן להעתקה)

### 1) לוגיקת ניתוב סוכן (TypeScript, משותף web/mobile)

```ts
// packages/core/agentRouter.ts
export type Tag = "mind"|"heart"|"body"|"action"|"soul";
const riskWords = [/להרביץ|להרוג|לא רוצה לחיות|סכין|אאבד את/i];

export function detectRisk(text: string){ return riskWords.some(r=>r.test(text)); }

export function classify(text: string): Tag {
  const t = text.toLowerCase();
  if (/(אני לא שווה|אני לא ראוי|תמיד|אף פעם)/.test(t)) return "soul";
  if (/(שונאים אותי|בושה|פחד|כעס|עצבות)/.test(t)) return "heart";
  if (/(כואב לי|בטן|לחץ בחזה|דופק)/.test(t)) return "body";
  if (/(אני הורס|לא מצליח|שלחתי סטורי|צחקתי עליו)/.test(t)) return "action";
  return "mind";
}

export function craftAffirmation(tag: Tag, text: string){
  switch(tag){
    case "heart": return "אני כועס/ת, ואני בוחר/ת לא לפגוע.";
    case "mind":  return "אני רואה את הסיפור יותר מאוזן, צעד-צעד.";
    case "body":  return "אני נושם/ת 4-4-6 והגוף נרגע לאט.";
    case "action":return "אני בוחר/ת התנהגות שמכבדת אותי ואת האחרים.";
    case "soul":  return "אני ראוי/ה לכבוד ואהבה – גם כשכואב.";
  }
}
```

## 3) אוטומציות (Flows) – לוגיקה ב-Lovable

### Flow: `detect_risk_words`

* **When**: onMessageSubmit (ב־/solo וב־/group)
* **If**: `message.text` מכיל מילה מתוך `scenes.riskKeywords`
* **Then**:

  * Save `riskFlag=true` בסשן
  * Redirect → `/safe`

### Flow: `route_agent`

* **When**: onMessageSubmit (לאחר בדיקת סיכון)
* **Classify** (כללי מפתח פשוטים):

  * אם הטקסט מכיל: “לא שווה/לא ראוי/תמיד/אף פעם” → `soul`
  * אם: “בושה/כעס/פחד/עצוב/שונאים אותי” → `heart`
  * אם: “בטן/לחץ/דופק/כאב בגוף” → `body`
  * אם: “שלחתי/צחקתי עליו/עשיתי/לא מצליח” → `action`
  * אחרת → `mind`
* **Then**: הצג `agents[tag].intro` → הצג `agents[tag].skill` → שמור `affirmation` → הצע מעבר ל־`/thanks`

### Flow: `group_room`

* **When**: onClick “יצירת חדר”
* **Do**: generate 6-char code (A-Z, 2-9), צור רשומת Room עם `sceneId`, `participants[]`
* **Join**: משתמשים מזינים קוד בחיבור → מצטרפים ל־Room

### Flow: `group_sentence_builder`

* כל משתתף שולח **מילה אחת**
* כאשר `count(words) >= מספר משתתפים` → הדבק למentence → הצג על המסך → הצע שמירה ושיתוף

* שלב 2: הצגת “מפת רגשות” (תגיות מהטקסטים).
* שלב 3: נשימת 4-4-6 משותפת (טיימר).
* שלב 4: “כל אחד מוסיף מילה אחת למשפט הקבוצתי.”
* תוצאה: הצגת המשפט המשותף + כפתור “שמור/שתף”.

### מנחה `/mentor`

* פתיחה רכה + מבנה מפגש קצר (כמו שנתתי לך קודם).
* כפתור “הפעל משחק קבוצתי” → ניתוב ל־`/group`.

### בטיחות `/safe`

* כותרת: “עצירה בטוחה 🌿”
* טקסט: “אם יש סכנה או מחשבות קשות – את/ה לא לבד.”
* קישורים (כפתורים גדולים):

  * “מוקד 105 – דיווח על פגיעה ברשת”
  * “ער״ן 1201 – סיוע רגשי”
  * “סה״ר – סיוע והקשבה ברשת”
* כפתור: “חזרה כשאני בטוח/ה”

### סיום `/thanks`

* כותרת: “יפה מאוד 🌿”
* “💎 המשפט המועיל שלך: **{affirmation}**”
* “צעד קטן להיום: למי בטוח תוכל/י לספר?”
* כפתורי שיתוף (אופציונלי)

## 5) רכיבי UI ממליצים (Lovable)

* **Chat**: קומפוננטת צ’אט מוכנה + “נשימה 4-4-6” כ-CTA מתחת לשדה הקלט.
* **Pills/Tags**: להצגת רגשות שעלו (בושה/כעס/פחד/חרטה/בלבול).
* **Progress**: פס רך להתקדמות (פתיחה → תרגול → משפט → סיום).
* **Modal**: קפיצה למסך בטיחות אם מזוהה סיכון.
* **Share**: שיתוף המשפט המועיל כתמונה (Canvas פשוט).

## 6) תוכן דמו “הסטורי” – דוגמאות מוכנות

**משפטי כאב אפשריים** → **מיפוי:**

* “כולם שונאים אותי” → HEART
* “אני לא שווה כלום” → SOUL
* “הלב דופק לי בטירוף” → BODY
* “שלחתי גם עליו, שיתבייש” → ACTION
* “תמיד זה קורה לי” → MIND/SOUL (בחירה לפי ניסוח)

**אישורים (Affirmations) מוכנים:**

* HEART: “אני כועס/ת, ואני בוחר/ת לא לפגוע.”
* MIND: “אני רואה את הסיפור בצורה מאוזנת.”
* BODY: “אני נושם/ת 4-4-6 והגוף נרגע לאט.”
* ACTION: “אני בוחר/ת התנהגות שמכבדת אותי ואת האחרים.”
* SOUL: “אני ראוי/ה לאהבה ולכבוד – גם כשכואב.”
