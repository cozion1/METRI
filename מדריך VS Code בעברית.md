## 🎯 פקודות נפוצות בטרמינל

### ניווט
```bash
cd שם_תיקייה    # היכנס לתיקייה
cd ..            # עלה תיקייה אחת למעלה
dir או ls        # הצג תוכן תיקייה
pwd              # איפה אני נמצא
```

### Python
```bash
python קובץ.py          # הרץ קובץ פייתון
pip install חבילה      # התקן חבילה
python --version        # בדוק גרסה
```

### Node.js
```bash
node קובץ.js           # הרץ קובץ JavaScript
npm install            # התקן תלויות
npm start              # הרץ פרויקט
```

### Git
```bash
git status             # בדוק מצב
git add .              # הוסף הכל
git commit -m "הודעה" # שמור גרסה
git push               # העלה לשרת
git pull               # הורד מהשרת
```

## 🆘 עזרה מהירה

### בעיה: הקובץ לא נשמר
✅ לחץ `Ctrl + S` או בדוק אם יש נקודה לבנה ליד שם הקובץ

### בעיה: הקוד לא רץ
✅ בדוק שיש לך טרמינל פתוח (``Ctrl + ` ``)
✅ ודא שאתה בתיקייה הנכונה (`cd` לתיקייה)

### בעיה: שגיאות אדומות בקוד
✅ עבור ל-View → Problems לראות את הבעיות
✅ לחץ על השגיאה כדי לקפוץ לשורה הבעייתית

### בעיה: איבדתי פאנל
✅ View → Appearance → Reset View Layout

## 🌟 טיפים מקצועיים

1. **Command Palette** (`Ctrl+Shift+P`) - הכלי החשוב ביותר! פה תמצא את כל הפקודות
2. **Multi-Cursor** - `Alt + Click` כדי להציב סמנים מרובים
3. **Format Document** - `Shift+Alt+F` לעיצוב אוטומטי של הקוד
4. **Duplicate Line** - `Shift+Alt+↓` לשכפול שורה
5. **Move Line** - `Alt+↑/↓` להזיז שורה למעלה/למטה
